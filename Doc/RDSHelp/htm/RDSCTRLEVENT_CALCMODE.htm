Б.2.3. RDSCTRLEVENT_CALCMODE – режим моделирования

Событие RDSCTRLEVENT_CALCMODE наступает при переходе RDS в режим моделирования. У этого события нет параметров и структуры описания. Реакция на него возможна следующими способами:

  • с помощью стандартной функции обратного вызова, зарегистрированной вызовом rdsctrlRegisterEventStdCallback, при этом в третьем параметре этой функции вместо указателя на структуру описания передается NULL;
  • с помощью сообщения окну управляющей программы, зарегистрированному функцией rdsctrlRegisterEventMessage, при этом в поле Data структуры RDSCTRL_MSGEVENTDATA, указатель на которую передается в параметре LPARAM оконного сообщения, вместо указателя на структуру описания события записано значение NULL.

См. также

Способы реакции на события, rdsctrlRegisterEventStdCallback, rdsctrlRegisterEventMessage, RDSCTRL_MSGEVENTDATA, RDSCTRLEVENT_EDITMODE.
