А.5.29.2. Команда RDS_BEU_STORECHANGED – запись измененных параметров блока

Команда RDS_BEU_STORECHANGED записывает измененные параметры блока в общий набор запомненных операций RDS.

Вызов команды для установки

  BOOL bOk=rdsCommandObject(Undo,RDS_BEU_STORECHANGED);

или

  BOOL bOk=rdsCommandObjectEx(Undo,RDS_BEU_STORECHANGED,0,NULL);

Параметры и результат

Undo (RDS_HOBJECT)
Идентификатор вспомогательного объекта, ранее созданного функцией rdsBEUCreate.
bOk (BOOL)
TRUE – параметры записаны, FALSE – ошибка (отмена операций запрещена в настройках RDS или блок, для параметров которого создан объект, удален).

Примечания

Эта команда записывает текущие значения параметров блока, идентификатор которого ранее был передан в функцию rdsBEUCreate при создании объекта, в набор запомненных операций RDS. После этого пользователь сможет отменить изменение параметров через пункт «Система | Отменить» главного меню RDS.

См. также

rdsBEUCreate, rdsCommandObject, rdsCommandObjectEx.
