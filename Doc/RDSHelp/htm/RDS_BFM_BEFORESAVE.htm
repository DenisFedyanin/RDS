А.2.5.3. RDS_BFM_BEFORESAVE – начато сохранение схемы

Поток, в котором вызывается функция модели

Главный поток RDS.

Первый параметр функции модели (int CallMode)

Константа RDS_BFM_BEFORESAVE.

Третий параметр функции модели (void *ExtParam)

Не используется (NULL).

Возвращаемое функцией модели значение

Не используется, можно возвращать любое значение.

Примечания

Реакция на это событие вызывается у всех блоков схемы непосредственно перед началом сохранения схемы в файл или передачей ее полного содержимого другому приложению через библиотеку дистанционного управления «RdsCtrl.dll» (см. §3.5 руководства программиста). В этой реакции можно подготовить к сохранению какие-либо параметры, относящиеся ко всей схеме в целом, или привести схему в состояние, пригодное для сохранения. Например, если блоки по командам пользователя вносили в схему какие-либо изменения, которые не должны сохраняться, в реакции на событие RDS_BFM_BEFORESAVE можно привести схему в исходный вид. Файл для сохранения схемы на момент вызова этой реакции еще не открыт, поэтому вызывать сервисные функции для сохранения данных блока в ней не следует (эти вызовы будут проигнорированы).
