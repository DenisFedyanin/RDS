А.3.4.11. RDS_COMPM_MODECHANGE – изменение режима RDS

Первый параметр функции модуля (int CallMode)

Константа RDS_COMPM_MODECHANGE.

Третий параметр функции модуля (void *ExtParam)

Не используется (NULL).

Возвращаемое функцией модуля значение

Не используется, можно возвращать любое значение.

Примечания

Событие RDS_COMPM_MODECHANGE возникает при любом изменении режима работы RDS. Обычно реакция на это событие используется для разрешения и запрещения различных органов управления в немодальных окнах редактора модели, если модуль создает такие окна (например, можно запретить изменение модели, если RDS не находится в режиме редактирования). Текущий режим работы RDS не передается в параметрах этого события, его можно узнать при помощи вызова сервисных функций rdsSystemInEditMode и rdsCalcProcessIsRunning.

См. также

RDS_COMPM_OPENEDITOR, rdsSystemInEditMode, rdsCalcProcessIsRunning.
