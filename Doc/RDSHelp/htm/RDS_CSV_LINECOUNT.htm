А.5.32.11. Команда RDS_CSV_LINECOUNT – число строк в тексте

Команда RDS_CSV_LINECOUNT возвращает общее число строк в тексте CSV.

Вызов команды

  int iCount=rdsGetObjectInt(Csv,RDS_CSV_LINECOUNT,0);

Параметр и результат

Csv (RDS_HOBJECT)
Идентификатор вспомогательного объекта для работы с CSV, ранее созданного функцией rdsCSVCreate.
iCount (int)
Число строк в тексте, включая пустые.

См. также

rdsCSVCreate, rdsGetObjectInt, rdsCSVGetItem, RDS_CSV_MAXCOLUMNS, RDS_CSV_LINECOLUMNS.
