А.5.26.7. Команда RDS_HSTR_GETLASTWORD – получить последнее считанное слово

Команда RDS_HSTR_GETLASTWORD возвращает последнее слово, считанное из текста функцией rdsSTRGetWord.

Вызов команды

  char *strWord=rdsGetObjectStr(Parser,RDS_HSTR_GETLASTWORD,0);

Параметры и результат

Parser (RDS_HOBJECT)
Идентификатор вспомогательного объекта для разбора текста, ранее созданного функцией rdsSTRCreateTextReader.
strWord (char*)
Указатель на строку во внутренней памяти объекта Vars, в которой записано слово, извлеченное из текста при последнем вызове для данного объекта функции rdsSTRGetWord.

Примечания

Указатель, возвращенный командой, можно использовать только до следующего вызова rdsSTRGetWord.

См. также

rdsSTRCreateTextReader, rdsGetObjectStr, rdsSTRGetWord.
