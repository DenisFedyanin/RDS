А.5.31.8. Команда RDS_PAN_LEFT – горизонтальная координата панели

Команда RDS_PAN_LEFT позволяет считывать и устанавливать горизонтальную координату левого верхнего угла панели.

Вызов команды для установки

  int iLeft= … // Координата
  rdsSetObjectInt(Panel,RDS_PAN_LEFT,0,iLeft);

Вызов команды для чтения

  int iLeft=rdsGetObjectInt(Panel,RDS_PAN_LEFT,0);

Параметры и результат

Panel (RDS_HOBJECT)
Идентификатор вспомогательного объекта-панели, ранее созданного функцией rdsPANCreate.
iLeft (int)
Горизонтальная координата левого верхнего угла панели в точках экрана в масштабе 100%.

Примечания

Вместе с командой RDS_PAN_TOP эта команда позволяет узнавать текущее положение панели и перемещать ее внутри окна подсистемы.

Пример

Пример использования команды приведен в §2.10.4 руководства программиста.

См. также

rdsPANCreate, rdsSetObjectInt, rdsGetObjectInt, RDS_PAN_TOP.
