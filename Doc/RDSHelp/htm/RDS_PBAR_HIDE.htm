А.5.30.3. Команда RDS_PBAR_HIDE – убрать индикатор с экрана

Команда RDS_PBAR_HIDE скрывает индикатор, относящийся к указанному объекту (сам объект при этом не уничтожается).

Вызов команды

  rdsCommandObject(Bar,RDS_PBAR_HIDE);

или

  rdsCommandObjectEx(Bar,RDS_PBAR_HIDE,0,NULL);

Параметр

Bar (RDS_HOBJECT)
Идентификатор вспомогательного объекта для работы с индикатором, ранее созданного функцией rdsPBARCreate.

Примечания

Для возврата скрытого индикатора на экран следует использовать команду RDS_PBAR_SHOW.

См. также

rdsPBARCreate, rdsCommandObject, rdsCommandObjectEx, RDS_PBAR_SHOW.
