Приложение В. Параметры командной строки RDS

В.2. «/calcmode» – перейти в режим моделирования

Если в командной строке RDS указан параметр «/calcmode», после загрузки схемы RDS перейдет в режим моделирования (по умолчанию RDS запускается в режиме редактирования). Без указания имени файла схемы в командной строке этот параметр не имеет смысла, поскольку переключать режимы RDS можно только при загруженной схеме.

Пример

  rds.exe "c:\work\rds\sample.rds" /calcmode

После запуска RDS загрузить схему «sample.rds» из папки «c:\work\rds» и перейти в режим моделирования.
