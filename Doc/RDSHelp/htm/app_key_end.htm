Приложение В. Параметры командной строки RDS

В.3. «/end» – прекратить разбор параметров

Если в командной строке указан параметр «/end», все следующие за ним параметры не будут восприняты RDS. Вместо этого они будут запомнены, и модели блоков смогут обращаться к ним при помощи сервисных функций rdsFindCmdParam, rdsGetCmdParam и rdsGetCmdParamCount. Так можно передать параметры непосредственно моделям блоков без обработки RDS.

Пример

  rds.exe "c:\work\rds\sample.rds" /end param1 param2 "param 3"

После запуска RDS загрузить схему «sample.rds» из папки «c:\work\rds» и запомнить параметры «param1», «param2» и «param 3» для выдачи их моделям блоков (функция rdsGetCmdParamCount в данном случае вернет число 3).
