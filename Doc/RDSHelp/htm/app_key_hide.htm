Приложение В. Параметры командной строки RDS

В.4. «/hide» – скрыть главное окно RDS

Если в командной строке указан параметр «/hide», главное окно RDS будет скрыто от пользователя. Роль главного окна будет выполнять окно корневой подсистемы загруженной схемы – при его закрытии RDS завершится.

При внешнем управлении через библиотеку «RdsCtrl.dll» (см. главу 3 руководства программиста) для скрытия главного окна RDS используется функция rdsctrlShowMainWindow.

Пример

  rds.exe "c:\work\rds\sample.rds" /hide

После запуска RDS загрузить схему «sample.rds» из папки «c:\work\rds» и скрыть главное окно.
