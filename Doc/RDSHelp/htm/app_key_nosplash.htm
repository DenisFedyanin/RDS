Приложение В. Параметры командной строки RDS

В.6. «/nosplash» – не выводить заставку RDS

Если в командной строке указан параметр «/nosplash», в процессе запуска RDS не будет выводиться стандартная заставка. Обычно этот параметр используют при включении RDS в состав каких-либо программных комплексов, имеющих собственную заставку.

Пример

  rds.exe /nosplash "c:\work\rds\sample.rds"

Запустить RDS, не выводя заставку, и загрузить схему «sample.rds» из папки «c:\work\rds».
