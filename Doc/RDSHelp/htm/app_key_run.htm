Приложение В. Параметры командной строки RDS

В.7. «/run» – перейти в режим расчета

Если в командной строке RDS указан параметр «/run», после загрузки схемы RDS запустит расчет загруженной схемы (по умолчанию RDS запускается в режиме редактирования). Без указания имени файла схемы в командной строке этот параметр не имеет смысла, поскольку переключать режимы RDS можно только при загруженной схеме.

Пример

  rds.exe "c:\work\rds\sample.rds" /run

После запуска RDS загрузить схему «sample.rds» из папки «c:\work\rds» и перейти в режим расчета.
