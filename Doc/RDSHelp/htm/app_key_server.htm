Приложение В. Параметры командной строки RDS

В.8. «/server» – запустить выделенный сервер RDS

Если в командной строке указан параметр «/server», RDS запустится в режиме выделенного сервера (см. §2.15.1 руководства программиста). Следующий за ним параметр трактуется как номер порта сервера. Если необходимо запустить сервер с номером порта по умолчанию, заданным в настройках RDS, вместо номера порта следует указать слово «default».

Примеры

  rds.exe /server 2345

Запустить RDS в режиме выделенного сервера, работающего через порт 2345.

  rds.exe /server default

Запустить RDS в режиме выделенного сервера, работающего через порт, указанный в настройках.
