А.5.29.3. Макрос rdsBEUStore – запись измененных параметров блока

Макрос rdsBEUStore записывает измененные параметры блока в общий набор запомненных операций RDS.

  rdsBEUStore(
      undo    // Вспомогательный объект
  )

Определение

  #define rdsBEUStore(undo) \
      rdsCommandObject((undo),RDS_BEU_STORECHANGED)

Параметр

undo (RDS_HOBJECT)
Идентификатор вспомогательного объекта, ранее созданного функцией rdsBEUCreate.

Возвращаемое значение

TRUE – параметры записаны, FALSE – ошибка (отмена операций запрещена в настройках RDS или блок, для параметров которого создан объект, удален).

Примечания

Этот макрос заключает в себя вызов команды RDS_BEU_STORECHANGED.
