А.5.2.9. rdsBringAppToFront – перемещение RDS на передний план

Функция rdsBringAppToFront перемещает приложение RDS на передний план, то есть окна RDS становятся видимыми поверх других окон Windows.

  void RDSCALL rdsBringAppToFront(void);

Тип указателя на эту функцию

RDS_VV

Примечания

Эта функция чаще всего применяется в моделях блоков, реагирующих на команды внешних приложений (см. главу 3 руководства программиста) для того, чтобы после каких-либо действий управляющего приложения вывести RDS на передний план и показать пользователю результат этих действий.

См. также

rdsctrlBringAppToFront.
