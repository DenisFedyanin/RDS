А.5.23.11. Макрос rdsCEClearEditor – очистка вспомогательного объекта

Макрос rdsCEClearEditor предназначен для очистки объекта редактирования связи.

  rdsCEClearEditor(
    editor    // Вспомогательный объект
  )

Определение

  #define rdsCEClearEditor(editor) \
      rdsCommandObject((editor),RDS_HCE_RESET)

Параметр

editor (RDS_HOBJECT)
Идентификатор вспомогательного объекта для редактирования связи/шины, ранее созданного функцией rdsCECreateEditor.

Примечания

Этот макрос заключает в себя вызов команды RDS_HCE_RESET, его можно использовать для того, чтобы сделать текст программы более понятным.

См. также
