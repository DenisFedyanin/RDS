А.5.28.28. Макрос rdsFORMClear – очистка объекта

Макрос rdsFORMClear удаляет из объекта все поля ввода, вкладки и боковую панель.

  rdsFORMClear(
      win    // Вспомогательный объект-окно
  )

Определение

  #define rdsFORMClear(win) \
      rdsCommandObject((win),RDS_FORM_CLEAR)

Параметр

win (RDS_HOBJECT)
Идентификатор вспомогательного объекта для работы с модальным окном, ранее созданного функцией rdsFORMCreate.

Примечания

Этот макрос заключает в себя вызов команды RDS_FORM_CLEAR.
