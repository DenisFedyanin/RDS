А.5.28.40. Макрос rdsFORMSetMultilineHeight – установка высоты многострочного поля ввода

Макрос rdsFORMSetMultilineHeight задает высоту в точках экрана для указанного поля ввода типа RDS_FORMCTRL_MULTILINE.

  rdsFORMSetMultilineHeight(
      win,       // Вспомогательный объект-окно
      ctrlid,    // Идентификатор поля ввода
      height     // Высота
  )

Определение

  #define rdsFORMSetMultilineHeight(win,ctrlid,height) \
      rdsSetObjectInt((win),(ctrlid),RDS_FORMVAL_MLHEIGHT,(height))

Параметры

win (RDS_HOBJECT)
Идентификатор вспомогательного объекта для работы с модальным окном, ранее созданного функцией rdsFORMCreate.
ctrlid (int)
Целый идентификатор поля ввода, присвоенный ему при вызове функции rdsFORMAddEdit.
height (int)
Высота поля ввода в точках экрана.

Примечания

Этот макрос заключает в себя вызов команды RDS_FORMVAL_MLHEIGHT с использованием функции rdsSetObjectInt.
