А.5.2.18. rdsGetAppWindowHandle – дескриптор главного окна RDS

Функция rdsGetAppWindowHandle возвращает дескриптор главного окна RDS, с которым должны быть связаны все остальные окна, открываемые в моделях и модулях автокомпиляции.

  HWND RDSCALL rdsGetAppWindowHandle(void);

Тип указателя на эту функцию

RDS_HwV

Возвращаемое значение

Дескриптор (HWND) главного окна приложения.

Примечания

Дескриптор окна, возвращаемый функцией rdsGetAppWindowHandle, обычно используется для привязки модальных окон, открываемых моделями блоков и модулями автоматической компиляции, к приложению RDS. Эта привязка выполняется при создании окон функциями Windows API, некоторые проблемы открытия модальных окон моделями блоков рассматривается в §1.8 и §2.2 руководства программиста.

Не следует путать эту функцию с функцией rdsGetMainWindow, которая возвращает дескриптор главного интерфейсного окна RDS, то есть окна с полосой меню, функциональными кнопками и вкладками панели блоков.

Пример

Пример использования функции rdsGetAppWindowHandle приведен в §2.7.5 руководства программиста.
