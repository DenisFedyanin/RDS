А.5.6.16. rdsGetBlockDescription – получить описание блока

Функция rdsGetBlockDescription заполняет структуру описания указанного блока.

  int RDSCALL rdsGetBlockDescription(
      RDS_BHANDLE Block,            // Блок
      RDS_PBLOCKDESCRIPTION pDescr  // Заполняемое описание
  );

Тип указателя на эту функцию

RDS_IBhBd

Параметры

Block (RDS_BHANDLE)
Идентификатор блока, описание которого нужно получить.
pDescr (RDS_PBLOCKDESCRIPTION)
Указатель на структуру описания блока RDS_BLOCKDESCRIPTION, которую функция должна заполнить.

Возвращаемое значение

1 – структура заполнена, 0 – ошибка.

Примечания

Функция заполняет структуру RDS_BLOCKDESCRIPTION основными параметрами блока. Для получения размеров блока в текущем масштабе следует использовать функцию rdsGetBlockDimensionsEx, для получения параметров окна подсистемы – функцию rdsGetEditorParameters.

Пример

Примеры использования функции rdsGetBlockDescription приведены в §2.6.3, §2.7.1, §2.16.2 и §4.3 руководства программиста, а также в некоторых других.

См. также

RDS_BLOCKDESCRIPTION, rdsGetBlockDimensionsEx, rdsGetEditorParameters.
