А.5.12.2. rdsGetBlockTimerDescr – получить описание таймера

Функция rdsGetBlockTimerDescr заполняет структуру описания указанного таймера.

  BOOL RDSCALL rdsGetBlockTimerDescr(
      RDS_TIMERID Timer               // Идентификатор таймера
      RDS_PTIMERDESCRIPTION pDescr    // Заполняемое описание
  );

Тип указателя на эту функцию

RDS_BThTd

Параметры

Timer (RDS_TIMERID)
Идентификатор таймера, описание которого нужно получить.
pDescr (RDS_PTIMERDESCRIPTION)
Указатель на структуру описания таймера RDS_TIMERDESCRIPTION, которую функция должна заполнить.

Возвращаемое значение

TRUE – структура заполнена, FALSE – ошибка.

Примечания

Функция заполняет структуру RDS_TIMERDESCRIPTION описанием таймера Timer.

Пример

Пример использования функции rdsGetBlockTimerDescr приведен в §2.9.2 руководства программиста.

См. также

RDS_TIMERDESCRIPTION, rdsSetBlockTimer.
