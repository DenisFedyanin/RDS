А.5.8.6. rdsGetEditorParameters – получить описание окна подсистемы

Функция rdsGetEditorParameters заполняет структуру описания окна указанной подсистемы.

  BOOL RDSCALL rdsGetEditorParameters(
      RDS_BHANDLE System,             // Подсистема
      RDS_PEDITORPARAMETERS pDescr    // Заполняемое описание
  );

Тип указателя на эту функцию

RDS_BBhEd

Параметры

System (RDS_BHANDLE)
Идентификатор подсистемы, описание окна которой нужно получить. Само окно при этом может быть закрыто, его описание хранится в параметрах подсистемы независимо от его наличия на экране.
pDescr (RDS_PEDITORPARAMETERS)
Указатель на структуру описания окна RDS_EDITORPARAMETERS, которую функция должна заполнить.

Возвращаемое значение

TRUE – структура заполнена, FALSE – ошибка (System – не подсистема, или размер заполняемой структуры не соответствует ожиданиям функции).

Пример

Примеры использования этой функции описаны в §2.10.4 и §2.12.5 руководства программиста.

См. также

RDS_EDITORPARAMETERS, rdsGetEditorFont, rdsGetEditorWindowFlags.
