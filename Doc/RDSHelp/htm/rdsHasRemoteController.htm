А.5.21.4. rdsHasRemoteController – проверка наличия внешнего управления

Функция rdsHasRemoteController проверяет, находится ли RDS под управлением внешней программы.

  BOOL RDSCALL rdsHasRemoteController(void);

Тип указателя на эту функцию

RDS_BV

Возвращаемое значение

TRUE – RDS в данный момент управляется внешним приложением, FALSE – RDS работает самостоятельно.

См. также

rdsGetRemoteControllerName.
