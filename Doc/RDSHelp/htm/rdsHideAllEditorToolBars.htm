А.5.8.11. rdsHideAllEditorToolBars – скрыть все панели окна подсистемы

Функция rdsHideAllEditorToolBars скрывает все панели окна указанной подсистемы. Вместо нее можно также использовать вызов rdsSetEditorWindowFlags с параметрами 0 и RDS_EWF_ALLBARS.

  void RDSCALL rdsHideAllEditorToolBars(
      RDS_BHANDLE System    // Подсистема
  );

Тип указателя на эту функцию

RDS_VBh

Параметр

System (RDS_BHANDLE)
Идентификатор подсистемы, панели окна которой нужно скрыть (окно подсистемы не обязательно должно быть открыто).

См. также

rdsSetEditorWindowFlags.
