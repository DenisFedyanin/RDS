А.5.27.25. Макрос rdsINISaveBlockText – передать текст параметров блока в RDS

Макрос rdsINISaveBlockText передает в RDS текст, содержащийся в объекте, для добавления к сохраняемым параметрам блока при реакции модели этого блока на событие RDS_BFM_SAVETXT.

  rdsINISaveBlockText(
    ini,       // Вспомогательный объект
    newline    // Перевод строки перед текстом
  )

Определение

  #define rdsINISaveBlockText(ini,newline) \
      rdsCommandObjectEx((ini),RDS_HINI_SAVEBLOCKTEXT, \
        (newline)?1:0,NULL)

Параметры

ini (RDS_HOBJECT)
Идентификатор вспомогательного объекта для работы с текстом, ранее созданного функцией rdsINICreateTextHolder.
newline (логическое: BOOL,int)
Логическое или целое значение, указывающее на необходимость добавить перед текстом объекта перевод строки (1, TRUE) или передать текст в RDS без изменений (0, FALSE).

Примечания

Этот макрос заключает в себя вызов команды RDS_HINI_SAVEBLOCKTEXT.
