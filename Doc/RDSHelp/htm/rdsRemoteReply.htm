А.5.21.6. rdsRemoteReply – возврат строки управляющему приложению

Функция rdsRemoteReply служит для указания строки, которую нужно вернуть управляющему приложению при реакции модели блока на событие RDS_BFM_REMOTEMSG.

  void RDSCALL rdsRemoteReply(
    LPSTR String  // Возвращаемая строка
  );

Тип указателя на эту функцию

RDS_VS

Параметр

String (LPSTR)
Указатель на строку, которую нужно вернуть управляющему приложению после завершения модели блока.

Примечания

Эту функцию следует вызывать из функции модели блока в момент реакции на событие RDS_BFM_REMOTEMSG. Вызов ее из любых других реакций будет игнорироваться. С помощью этой функции модель передает в RDS строку, которая, после завершения реакции, будет отправлена в управляющее приложение, где она будет использована в качестве результата выполнения функции rdsctrlCallBlockFunction или rdsctrlCallBlockFunctionEx. Если модель блока не вызовет rdsRemoteReply, управляющему приложению будет передана пустая строка.

Пример

Примеры использования функции rdsRemoteReply приведены в §3.3 руководства программиста и в описании события RDS_BFM_REMOTEMSG.

См. также

RDS_BFM_REMOTEMSG, rdsctrlCallBlockFunction, rdsctrlCallBlockFunctionEx.
