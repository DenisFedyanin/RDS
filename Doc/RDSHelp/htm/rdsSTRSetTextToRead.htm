А.5.26.16. Макрос rdsSTRSetTextToRead – получить из текста слово как целое число

Макрос rdsSTRSetTextToRead передает в объект текст, из которого функция rdsSTRGetWord будет последовательно извлекать слова.

  rdsSTRSetTextToRead(
    parser,    // Вспомогательный объект
    text       // Текст для разбора
  )

Определение

  #define rdsSTRSetTextToRead(parser,text) \
      rdsSetObjectStr((parser),RDS_HSTR_SETTEXT,0,(text))

Параметры

parser (RDS_HOBJECT)
Идентификатор вспомогательного объекта для разбора текста, ранее созданного функцией rdsSTRCreateTextReader.
text (char*)
Указатель на текст, который будет разбираться объектом.

Возвращаемое значение

Примечания

Этот макрос заключает в себя вызов команды RDS_HSTR_SETTEXT.
