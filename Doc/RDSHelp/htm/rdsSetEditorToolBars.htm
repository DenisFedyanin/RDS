А.5.8.17. rdsSetEditorToolBars – задать состояние панелей окна подсистемы (устаревшая)

Функция rdsSetEditorToolBars устанавливает видимость отдельных панелей окна подсистемы согласно структуре RDS_EDITORTOOLBARS. Это устаревшая функция, сейчас вместо нее используется rdsSetEditorWindowFlags.

  void RDSCALL rdsSetEditorToolBars(
      RDS_BHANDLE System,          // Подсистема
      RDS_PEDITORTOOLBARS pBars    // Структура описания
  );

Тип указателя на эту функцию

RDS_VBhEt

Параметры

System (RDS_BHANDLE)
Идентификатор подсистемы, состояние панелей окна которой нужно установить (окно подсистемы не обязательно должно быть открыто).
pBars (RDS_PEDITORTOOLBARS)
Указатель на структуру описания, согласно которой функция установит видимость панелей.

Примечания

Эта функция устарела и позволяет установить видимость только некоторых панелей окна.

См. также

RDS_EDITORTOOLBARS, rdsSetEditorWindowFlags.
