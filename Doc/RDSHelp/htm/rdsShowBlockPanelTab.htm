А.5.2.39. rdsShowBlockPanelTab – управление вкладками панели блоков

Функция rdsShowBlockPanelTab позволяет включать и выключать видимость отдельных вкладок панели блоков.

  void RDSCALL rdsShowBlockPanelTab(
      int Op,        // Операция с вкладкой
      LPSTR TabName  // Имя вкладки
  );

Тип указателя на эту функцию

RDS_VIS

Параметры

Op (int)
Действие, выполняемое с вкладкой, имя которой передано в параметре TabName. Может принимать одно из следующих значений:
RDS_BLOCKPANELOP_HIDE Скрыть вкладку TabName.
RDS_BLOCKPANELOP_SHOW Показать вкладку TabName.
RDS_BLOCKPANELOP_HIDEEXCEPT Скрыть все вкладки кроме TabName.
RDS_BLOCKPANELOP_SHOWALL Показать все вкладки.
RDS_BLOCKPANELOP_SELECT Сделать вкладку TabName текущей.
TabName (LPSTR)
Указатель на строку с именем вкладки (имя вкладки панели блоков – это имя папки, в которой находятся файлы ее блоков, см. §2.16.4 описания пользователя и §1.1 руководства программиста).

Примечания

Управление вкладками панели блоков обычно используется в тех случаях, когда RDS входит в состав другого приложения и используется как редактор каких-либо схем. При этом пользователю обычно показывают только те вкладки, на которых размещаются блоки, которые можно добавлять в редактируемую схему.
