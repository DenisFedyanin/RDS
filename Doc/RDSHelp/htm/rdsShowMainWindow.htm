А.5.2.40. rdsShowMainWindow – управление видимостью главного окна RDS

Функция rdsShowMainWindow позволяет скрыть или показать пользователю главное окно RDS (окно с полосой меню, функциональными кнопками и панелью блоков).

  void RDSCALL rdsShowMainWindow(
      BOOL Show    // Показать/скрыть
  );

Тип указателя на эту функцию

RDS_VB

Параметр

Show (BOOL)
TRUE, если главное окно должно быть видимо пользователю, или FALSE, если оно должно быть скрыто.

Примечания

Исходное состояние главного окна определяется параметрами командной строки RDS (запуск с параметром «/hide» делает его скрытым) или управляющим приложением, если RDS работает под его управлением. По умолчанию главное окно видимо.

При скрытом главном окне закрытие окна корневой подсистемы завершает RDS.

См. также

rdsMainWindowVisible, параметры командной строки RDS.
