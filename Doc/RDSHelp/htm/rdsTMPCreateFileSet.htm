А.5.19.2. rdsTMPCreateFileSet – создать набор временных файлов

Функция rdsTMPCreateFileSet создает в памяти RDS список временных файлов, в который можно добавлять файлы по одному, а затем, когда они станут не нужны, удалить все файлы из списка одним вызовом.

  int RDSCALL rdsTMPCreateFileSet(void);

Тип указателя на эту функцию

RDS_IV

Возвращаемое значение

Уникальный идентификатор набора временных файлов, который можно использовать в других функциях работы с временными файлами.

Примечания

Эта функция создает набор (внутренний список) временных файлов и возвращает его идентификатор. Файлы в набор можно добавлять функциями rdsTMPCreateEmptyFile и rdsTMPRememberFileName. Когда набор временных файлов станет не нужен, его можно удалить функцией rdsTMPDeleteFileSet, при этом сами файлы из этого набора тоже будут удалены с диска. При выгрузке схемы из памяти автоматически стираются все временные файлы во всех созданных за время работы схемы наборах.

Пример

Пример использования функции rdsTMPCreateFileSet приведен в §4.4 руководства программиста.

См. также

rdsTMPDeleteFileSet, rdsTMPCreateEmptyFile, rdsTMPRememberFileName.
