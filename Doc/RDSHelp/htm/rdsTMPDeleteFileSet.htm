А.5.19.4. rdsTMPDeleteFileSet – удалить набор временных файлов

Функция rdsTMPDeleteFileSet удаляет ранее созданный набор временных файлов из памяти и стирает с диска все файлы, принадлежащие этому набору.

  void RDSCALL rdsTMPDeleteFileSet(
    int SetId    // Идентификатор набора
  );

Тип указателя на эту функцию

RDS_VI

Параметр

SetId (int)
Уникальный целый идентификатор удаляемого набора временных файлов.

Примечания

Эта функция стирает все файлы набора временных файлов с идентификатором SetId, после чего удаляет этот набор из памяти.

Пример

Пример использования функции rdsTMPCreateFileSet приведен в §4.4 руководства программиста.

rdsTMPCreateFileSet, rdsTMPCreateEmptyFile, rdsTMPRememberFileName.
