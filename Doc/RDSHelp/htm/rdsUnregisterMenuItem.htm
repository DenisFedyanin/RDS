А.5.17.10. rdsUnregisterMenuItem – удалить постоянный пункт меню

Функция rdsUnregisterMenuItem удаляет ранее созданный постоянный пункт системного меню RDS или контекстного меню блока.

  void RDSCALL rdsUnregisterMenuItem(
    RDS_MENUITEM Item    // Идентификатор пункта меню
  );

Тип указателя на эту функцию

RDS_VMh

Параметры

Item (RDS_MENUITEM)
Уникальный идентификатор удаляемого пункта меню (RDS_MENUITEM).

Примечания

Эта функция удаляет пункт меню (не важно, контекстного или системного) с идентификатором Item.

Пример

Пример использования функции приведен в §2.12.6 и §2.12.7 руководства программиста.

См. также

rdsRegisterContextMenuItemEx, rdsRegisterContextMenuItem, rdsRegisterMenuItem.
