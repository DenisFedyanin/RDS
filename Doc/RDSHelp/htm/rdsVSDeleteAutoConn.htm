А.5.25.36. Макрос rdsVSDeleteAutoConn – удалить связь между основной и управляющей переменными

Макрос rdsVSDeleteAutoConn предназначен для удаления внутренней связи основной переменной со вспомогательной в объекте-редакторе переменных.

  rdsVSDeleteAutoConn(
    vars,      // Вспомогательный объект
    num        // Номер связи
  )

Определение

  #define rdsVSDeleteAutoConn(vars,num) \
      rdsCommandObjectEx((vars),RDS_HVAR_DELAUTO,(num),NULL)

Параметры

vars (RDS_HOBJECT)
Идентификатор вспомогательного объекта-редактора переменных, ранее созданного функцией rdsVSCreateEditor.
num (целое)
Целое число – номер удаляемой связи переменных.

Примечания

Этот макрос заключает в себя вызов команды RDS_HVAR_DELAUTO.
