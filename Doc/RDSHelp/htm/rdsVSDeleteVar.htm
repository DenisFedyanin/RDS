А.5.25.37. Макрос rdsVSDeleteVar – удалить переменную

Макрос rdsVSDeleteVar предназначен для удаления переменной с заданным номером из объекта-редактора.

  rdsVSDeleteVar(
    vars,      // Вспомогательный объект
    num        // Номер переменной
  )

Определение

  #define rdsVSDeleteVar(vars,num) \
      rdsCommandObjectEx((vars),RDS_HVAR_DELVAR,(num),NULL)

Параметры

vars (RDS_HOBJECT)
Идентификатор вспомогательного объекта-редактора переменных, ранее созданного функцией rdsVSCreateEditor.
num (целое)
Целое число – номер удаляемой переменной.

Примечания

Этот макрос заключает в себя вызов команды RDS_HVAR_DELVAR.
