А.5.18.7. rdsXGFillRect – заполненный прямоугольник

Функция rdsXGFillRect заполняет указанный прямоугольник текущей выбранной заливкой, если модель блока вызвана для рисования.

  void RDSCALL rdsXGFillRect(
    int Left,int Top,       // Левый верхний угол
    int Right,int Bottom    // Правый нижний угол
  );

Тип указателя на эту функцию

RDS_VIIII

Параметры

Left, Top (int)
Левый верхний угол (Left – горизонтальная координата, Top – вертикальная) закрашиваемой прямоугольной области.
Right, Bottom (int)
Правый нижний угол (Right – горизонтальная координата, Bottom – вертикальная) прямоугольной области.

Примечания

Эта функция закрашивает прямоугольную область (Left,Top) – (Right,Bottom) цветом и стилем, выбранными функцией rdsXGSetBrushStyle. От функции рисования прямоугольника rdsXGRectangle она отличается тем, что изображаемый ей прямоугольник не имеет рамки независимо от текущих установок стиля линии. Запомненная текущая точка рисования не изменяется.

Пример

Пример использования функции rdsXGFillRect приведен в §2.7.3 и §2.10.1 руководства программиста.

См. также

rdsXGSetBrushStyle, rdsXGRectangle, rdsXGInvertRect.
