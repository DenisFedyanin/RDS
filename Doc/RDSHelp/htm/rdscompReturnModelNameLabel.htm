А.5.34.10. rdscompReturnModelNameLabel – заголовок поля ввода имени модели

Функция rdscompReturnModelNameLabel передает в RDS заголовок поля ввода имени автокомпилируемой модели, указываемый в окне параметров блока.

  void RDSCALL rdscompReturnModelNameLabel(
    LPSTR ModelNameLabel    // Имя поля ввода
  );

Тип указателя на эту функцию

RDS_VS

Параметр

ModelNameLabel (LPSTR)
Указатель на строку с заголовком поля ввода имени модели.

Примечания

Эта функция может вызываться только из функции модуля автокомпиляции при реакции на события реакции на действия пользователя RDS_COMPM_EXECFUNCTION и запроса описания возможностей модуля RDS_COMPM_GETOPTIONS. В этих реакциях модуль сообщает RDS название, которое указывается на вкладке «компиляция» окна параметров блока перед полем ввода имени модели. Например, если модели хранятся в файлах, модуль может установить в качестве заголовка поля строку «Файл модели:».

Пример

Пример использования этой функции рассматривается в §4.3 руководства программиста.

См. также

RDS_COMPM_EXECFUNCTION, RDS_COMPM_GETOPTIONS.
