Б.3.4.5. rdsctrlBringAppToFront – переместить RDS на передний план

Функция rdsctrlBringAppToFront перемещает на передний план копию RDS, управляемую через указанную связь.

  void RDSCALL rdsctrlBringAppToFront(
    int Link    // Идентификатор связи
  );

Тип указателя на эту функцию

RDSCTRL_VI

Параметр

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.

Примечания

Эта функция располагает окна управляемой через связь Link копии RDS на переднем плане, поверх других приложений. Если по команде от внешнего приложения RDS будет открывать какое-либо окно с запросом пользователю, перед этим приложение RDS («rds.exe») необходимо переместить на передний план, чтобы пользователь увидел этот запрос.

См. также

rdsctrlCreateLink, rdsctrlMinimizeApp, rdsctrlRestoreApp, rdsctrlShowMainWindow.
