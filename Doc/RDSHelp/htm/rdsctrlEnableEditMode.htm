Б.3.3.2. rdsctrlEnableEditMode – разрешение режима редактирования

Функция rdsctrlEnableEditMode разрешает или запрещает пользователю переключать RDS в режим редактирования.

  void RDSCALL rdsctrlEnableEditMode(
    int Link,      // Идентификатор связи
    BOOL Enable    // Разрешение
  );

Тип указателя на эту функцию

RDSCTRL_VIB

Параметры

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.
Enable (BOOL)
TRUE – режим редактирования разрешен, FALSE – запрещен.

Примечания

Эта функция разрешает (при Enable==TRUE) или запрещает (при Enable==FALSE) пользователю включать режим редактирования в копии RDS, управляемой через связь Link (по умолчанию режим редактирования разрешен). Если в момент запрещения режима редактирования RDS находится в нем, будет автоматически включен режим моделирования.

Разрешение режима редактирования запоминается в параметрах связи Link, его не нужно устанавливать заново при повторном запуске RDS через эту связь. Можно, например, вызвать rdsctrlEnableEditMode(Link,FALSE) сразу после создания связи, когда еще нет работающей копии RDS. При этом после вызова rdsctrlConnect RDS запустится уже с запрещенным режимом редактирования и автоматически переключится в режим моделирования после запуска.

См. также

rdsctrlCreateLink, rdsctrlEnableCalcMode, rdsctrlEnableRun, rdsctrlSetGeneralSettings, rdsctrlConnect.
