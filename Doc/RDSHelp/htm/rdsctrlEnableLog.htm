Б.3.8.2. rdsctrlEnableLog – включить/выключить журнал

Функция rdsctrlEnableLog разрешает или запрещает ведение файла журнала, в которой библиотека «RdsCtrl.dll» записывает служебную информацию о связи с RDS и обнаруженных ошибках.

  void RDSCALL rdsctrlEnableLog(
    BOOL Enable,    // Разрешить/запретить
    BOOL Clear      // Очистить журнал
  );

Тип указателя на эту функцию

RDSCTRL_VBB

Параметры

Enable (BOOL)
TRUE – разрешить ведение журнала, FALSE – запретить.
Clear (BOOL)
TRUE – очистить файл журнала, FALSE – не очищать.

Примечания

При включенном журнале в текстовый файл, имя которого задано функцией rdsctrlSetLogFile, будут записываться служебные сообщения о ходе установки связи управляющего приложения с RDS и о возникающих ошибках. В журнал можно также добавлять пользовательские сообщения функцией rdsctrlLogString.

По умолчанию ведение журнала запрещено.

См. также

rdsctrlSetLogFile, rdsctrlLogString.
