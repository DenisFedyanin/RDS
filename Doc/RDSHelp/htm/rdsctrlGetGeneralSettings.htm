Б.3.3.9. rdsctrlGetGeneralSettings – получить настройки интерфейса пользователя RDS

Функция rdsctrlGetGeneralSettings возвращает общие настройки интерфейса пользователя RDS.

  void RDSCALL rdsctrlGetGeneralSettings(
    int Link,                      // Идентификатор связи
    RDSCTRL_PSETTINGS pSettings    // Настройки
  );

Тип указателя на эту функцию

RDSCTRL_VIOpt

Параметры

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.
pSettings (RDSCTRL_PSETTINGS)
Указатель на структуру описания общих параметров RDS RDSCTRL_SETTINGS, в которую функция запишет параметры, запомненные для связи Link.

Примечания

Эта функция заполняет структуру, указатель на которую передан в параметре pSettings, запомненными для связи Link настройками интерфейса пользователя RDS. Управляемая через эту связь копия RDS не обязательно должна быть запущена, эти параметры хранятся во внутренней памяти библиотеки RdsCtrl.dll для каждой созданной связи.

См. также

rdsctrlCreateLink, RDSCTRL_SETTINGS, rdsctrlSetGeneralSettings.
