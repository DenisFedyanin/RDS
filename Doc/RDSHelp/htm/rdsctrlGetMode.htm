Б.3.4.19. rdsctrlGetMode – получить режим работы RDS

Функция rdsctrlGetMode возвращает режим, в котором в данный момент находится управляемая копия RDS.

  int RDSCALL rdsctrlGetMode(
    int Link    // Идентификатор связи
  );

Тип указателя на эту функцию

RDSCTRL_II

Параметр

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.

Возвращаемое значение

Одна из констант RDSCTRL_MODE_*:

RDSCTRL_MODE_UNKNOWN Нет управляемой копии RDS.
RDSCTRL_MODE_EDIT RDS в режиме редактирования.
RDSCTRL_MODE_CALC RDS в режиме моделирования.
RDSCTRL_MODE_RUNNING RDS в режиме расчета.

Примечания

Эта функция запрашивает текущий режим у копии RDS, управляемой через связь Link. Если приложение «rds.exe» для этой связи не запущено, возвращается константа RDSCTRL_MODE_UNKNOWN.

См. также

rdsctrlCreateLink, rdsctrlSetEditMode, rdsctrlSetCalcMode, rdsctrlStartCalc.
