Б.3.5.6. rdsctrlLoadSystemFromFile – загрузить схему из файла

Функция rdsctrlLoadSystemFromFile загружает в управляемую копию RDS схему из файла с указанным именем.

  BOOL RDSCALL rdsctrlLoadSystemFromFile(
    int Link,          // Идентификатор связи
    LPSTR FileName,    // Имя файла
    BOOL SavePrompt    // Предложить сохранение
  );

Тип указателя на эту функцию

RDSCTRL_BISB

Параметры

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.
FileName (LPSTR)
Указатель на строку с полным путем к загружаемому файлу.
SavePrompt (BOOL)
TRUE – предлагать пользователю сохранить загруженную в данный момент схему, если в ней были изменения, перед загрузкой новой схемы из файла FileName. FALSE – выгрузить старую схему из памяти без запросов пользователю, не сохраняя ее.

Возвращаемое значение

TRUE – загрузка выполнена успешно, FALSE – при загрузке возникли ошибки.

Примечания

Эта функция загружает новую схему в копию RDS, управляемую через связь Link, из файла FileName.

Пример

Пример использования функции приведен в §3.2 руководства программиста.

rdsctrlCreateLink, rdsctrlSaveSystemToFile.
