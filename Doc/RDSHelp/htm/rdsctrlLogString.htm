Б.3.8.3. rdsctrlLogString – записать текст в журнал

Функция rdsctrlLogString записывает указанный текст в файл журнала библиотеки «RdsCtrl.dll».

  void RDSCALL rdsctrlLogString(
    LPSTR String,    // Текст
    BOOL CrLf        // Перевести строку
  );

Тип указателя на эту функцию

RDSCTRL_VSB

Параметры

String (LPSTR)
Указатель на текст, который нужно записать в журнал.
CrLf (BOOL)
TRUE – перевести строку после записи текста, FALSE – не переводить.

Примечания

Имя файла журнала задается функцией rdsctrlSetLogFile. В него записывается не только текст, переданный функцией rdsctrlLogString, но и служебные сообщения «RdsCtrl.dll». Для записи текста ведение журнала должно быть разрешено функцией rdsctrlEnableLog.

См. также

rdsctrlSetLogFile, rdsctrlEnableLog.
