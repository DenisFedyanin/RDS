Б.3.4.24. rdsctrlResetCalc – сбросить расчет

Функция rdsctrlResetCalc сбрасывает расчет в копии RDS, управляемой через указанную связь.

  void RDSCALL rdsctrlResetCalc(
    int Link    // Идентификатор связи
  );

Тип указателя на эту функцию

RDSCTRL_VI

Параметр

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.

Примечания

Эта функция сбрасывает расчет в копии RDS, управляемой через связь Link, то есть переводит всю схему в исходное состояние.

См. также

rdsctrlCreateLink, rdsctrlStartCalc, rdsctrlStopCalc.
