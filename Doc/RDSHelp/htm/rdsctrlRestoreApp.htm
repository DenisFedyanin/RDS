Б.3.4.25. rdsctrlRestoreApp – восстановить свернутое приложение RDS

Функция rdsctrlRestoreApp разворачивает ранее свернутую управляемую копию RDS.

  void RDSCALL rdsctrlRestoreApp(
    int Link    // Идентификатор связи
  );

Тип указателя на эту функцию

RDSCTRL_VI

Параметр

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.

Примечания

Эта функция восстанавливает свернутую копию RDS, управляемую через связь Link. Для сворачивания приложения следует использовать функцию rdsctrlMinimizeApp.

См. также

rdsctrlCreateLink, rdsctrlMinimizeApp, rdsctrlBringAppToFront.
