Б.3.2.8. rdsctrlRestoreConnection – перезапустить RDS при необходимости

Функция rdsctrlRestoreConnection проверяет наличие запущенной копии RDS и перезапускает rds.exe, если такой копии нет.

  BOOL RDSCALL rdsctrlRestoreConnection(
    int Link    // Идентификатор связи
  );

Тип указателя на эту функцию

RDSCTRL_BI

Параметр

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.

Возвращаемое значение

TRUE – после вызова функции есть связь с процессом rds.exe, (процесс не завершался или успешно запущен заново), FALSE – нет связи с RDS.

Примечания

Эта функция запускает RDS, если в данный момент для связи Link нет работающего процесса rds.exe. Если процесс есть, функция не выполняет никаких действий и немедленно возвращает TRUE.

Пример

Пример использования функции rdsctrlRestoreConnection приведен в §3.2 руководства программиста.
