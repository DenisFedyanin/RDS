Б.3.2.9. rdsctrlSetPath – установить путь к RDS

Функция rdsctrlSetPath передает в библиотеку RdsCtrl.dll путь к исполняемому файлу RDS.

  void RDSCALL rdsctrlSetPath(
    LPSTR Path    // Путь к rds.exe
  );

Тип указателя на эту функцию

RDSCTRL_VS

Параметр

Path (LPSTR)
Указатель на строку с полным путем к исполняемому файлу RDS «rds.exe».

Примечания

Эта функция вызывается в тех случаях, когда «rds.exe» и библиотека RdsCtrl.dll находятся в разных папках. Функция не привязана к какой-либо конкретной связи с RDS, ее достаточно вызвать один раз сразу после загрузки библиотеки в память. Если RdsCtrl.dll и «rds.exe» находятся в одной папке, вызывать rdsctrlSetPath не обязательно.
