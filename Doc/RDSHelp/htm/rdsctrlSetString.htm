Б.3.4.31. rdsctrlSetString – установить глобальную строку

Функция rdsctrlSetString устанавливает строку с идентификатором, по которому модели блоков могут обращаться к этой строке.

  void RDSCALL rdsctrlSetString(
    int Link,       // Идентификатор связи
    int Id,         // Идентификатор строки
    LPSTR String    // Строка
  );

Тип указателя на эту функцию

RDSCTRL_VIIS

Параметры

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.
Id (int)
Целый идентификатор устанавливаемой строки.
String (LPSTR)
Указатель на текст устанавливаемой строки.

Примечания

Эта функция устанавливает в копии RDS, управляемой через связь Link, строку с идентификатором Id и текстом String. Модель любого блока схемы, загруженной в эту копию RDS, может обратиться к строке по идентификатору при помощи функции rdsGetRemoteControllerString. Например, если управляющее приложение сделает вызов rdsctrlSetString(Link,23,"abcd"), модель блока, сделав вызов rdsGetRemoteControllerString(23), получит указатель на строку «abcd».

Набор строк с идентификаторами запоминается в параметрах связи Link, эти строки не нужно устанавливать заново при повторном запуске RDS через эту связь. Можно, например, сделать все вызовы rdsctrlSetString один раз сразу после создания связи, когда еще нет работающей копии RDS. При этом после вызова rdsctrlConnect все эти строки будут переданы в RDS автоматически.

См. также

rdsctrlCreateLink, rdsGetRemoteControllerString, rdsctrlConnect.
