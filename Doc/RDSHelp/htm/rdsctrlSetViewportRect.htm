Б.3.7.7. rdsctrlSetViewportRect – задать положение порта вывода в окне

Функция rdsctrlSetViewportRect устанавливает координаты прямоугольной области, занимаемой портом вывода в окне управляющего приложения.

  void RDSCALL rdsctrlSetViewportRect(
    int Link,             // Идентификатор связи
    int VpId,             // Идентификатор порта вывода
    int Left,int Top,     // Левый верхний угол
    int Width,int Height  // Размеры
  );

Тип указателя на эту функцию

RDSCTRL_VIIIIII

Параметры

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.
VpId (int)
Идентификатор порта вывода, присвоенный ему при создании функцией rdsctrlSetViewport.
Left, Top (int)
Горизонтальная (Left) и вертикальная (Top) координаты левого верхнего угла прямоугольной области, занимаемой портом вывода, в окне управляющего приложения.
Width, Height (int)
Ширина (Width) и высота (Height) порта вывода в окне управляющего приложения.

Примечания

Эта функция устанавливает координаты и размеры порта вывода VpId в копии RDS, управляемой через связь Link.

Пример

Пример использования функции приведен в §3.6.2 руководства программиста.
