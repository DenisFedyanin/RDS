Б.3.5.18. rdsctrlStartBlockByBlockLoad – начать поблочную загрузку схемы

Функция rdsctrlStartBlockByBlockLoad начинает поблочную загрузку схемы в управляемую копию RDS, при которой каждый блок, связь или шина схемы передаются в RDS отдельным вызовом.

  void RDSCALL rdsctrlStartBlockByBlockLoad(
    int Link,      // Идентификатор связи
    DWORD Flags    // Флаги (не используются)
  );

Тип указателя на эту функцию

RDSCTRL_VIDw

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.
Flags (DWORD)
Флаги функции (в настоящее время этот параметр никак не используется).

Примечания

Эта функция начинает поблочную передачу схемы в копию RDS, управляемую через связь Link. После ее вызова можно в произвольном порядке передавать в RDS описания различных объектов схемы функцией rdsctrlSetBlockByBlockLoadPiece, для завершения загрузки и сборки схемы из набора переданных объектов следует вызвать rdsctrlEndBlockByBlockLoad.

Пример

Пример использования функции rdsctrlStartBlockByBlockLoad приведен в §3.5 руководства программиста.

См. также

rdsctrlCreateLink, rdsctrlSetBlockByBlockLoadPiece, rdsctrlEndBlockByBlockLoad, rdsctrlStartBlockByBlockSave.
