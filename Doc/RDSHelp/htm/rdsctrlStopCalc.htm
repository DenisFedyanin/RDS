Б.3.4.33. rdsctrlStopCalc – остановить расчет

Функция rdsctrlStopCalc останавливает работающий расчет в управляемой копии RDS.

  void RDSCALL rdsctrlStopCalc(
    int Link    // Идентификатор связи
  );

Тип указателя на эту функцию

RDSCTRL_VI

Параметр

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.

Примечания

Если копия RDS, управляемая через связь Link, находится в режиме расчета, вызов этой функции останавливает расчет и переводит ее в режим моделирования. В режимах моделирования и редактирования вызов функции игнорируется. Остановка расчета требует некоторого времени, поэтому на момент завершения функции rdsctrlStopCalc расчет может еще работать.

Пример

Пример использования функции rdsctrlStopCalc приведен в §3.2 руководства программиста.
