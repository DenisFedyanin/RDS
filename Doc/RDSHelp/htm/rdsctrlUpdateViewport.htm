Б.3.7.10. rdsctrlUpdateViewport – обновить порт вывода

Функция rdsctrlUpdateViewport перерисовывает изображение указанного порта вывода.

  void RDSCALL rdsctrlUpdateViewport(
    int Link,    // Идентификатор связи
    int VpId     // Идентификатор порта вывода
  );

Тип указателя на эту функцию

RDSCTRL_VII

Параметры

Link (int)
Идентификатор связи с RDS, присвоенный ей при создании функцией rdsctrlCreateLink.
VpId (int)
Идентификатор порта вывода, присвоенный ему при создании функцией rdsctrlSetViewport.

Примечания

Эта функция дает команду обновить изображение подсистемы в порте вывода VpId в копии RDS, управляемой через связь Link, и ожидает окончания этого обновления.

Пример

Пример использования функции rdsctrlUpdateViewport приведен в §3.6.2 руководства программиста.

См. также

rdsctrlCreateLink, rdsctrlSetViewport.
